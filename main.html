Dokumentasi Model & Formula Matematis

Penjelasan variabel dan rumus yang digunakan untuk simulasi Uji Stres (Stress Test).

Kamus Variabel (Variables)

I = Pendapatan/Omset bulanan
S = % Income Shock
E_fixed = Pengeluaran bulanan tetap
E_var = Pengeluaran variabel / HPP
A_cash = Kas / Tabungan cair
A_inv = Nilai investasi
A_phys = Nilai aset fisik
D = Total utang pokok
Q = Kuantitas inventaris
P = Harga beli per unit
pi = Tingkat inflasi (%)
r_bi = Suku bunga BI (%)

Rumus Kalkulasi (Formulas)

  • I_stress = I × (1 - S / 100)
  • E_total = E_fixed + E_var
  • Surplus = I_stress - E_total
  • Inventory = sum(Q × P)
  • A_liquid = A_cash + A_inv
  • A_total = A_liquid + A_phys + Inventory
  • NetWorth = A_total - D
  • DebtRatio = D / (I_stress × 12)
  • If Surplus < 0: MonthlyBurn = abs(Surplus)
    If Surplus ≥ 0: MonthlyBurn = E_total
  • CashRunway = A_liquid / MonthlyBurn
  • RealPurchasingPower = I_stress / (1 + pi / 100)
